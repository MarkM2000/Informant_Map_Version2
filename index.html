Informant Map Version 2

About Map

This is Version 2 of the Informant Map project. There are checkboxes on this map that correspond to the state informants. The state informant, gender, ethnicity, occupation, education, and fieldworker are among the checkboxes. The projects are LAGS, LAMSAS, LANCS, and LANE.
